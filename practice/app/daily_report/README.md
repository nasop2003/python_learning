# 日報ジェネレーター

## 概要
- 日報をtxtファイルに書き込むアプリです
- 「今日やったこと」「学んだこと」「明日やること」の3つを入力できます（学んだこと・明日やることは空白でも可能です）

## 機能
- 操作したい処理（日報作成・削除）を番号で選択
- 3項目（今日やったこと・学んだこと・明日やること）をターミナルで入力
- 削除したい日報を番号で選択

## 使い方
1. `daily_report.py`を実行
2. メイン画面が表示され、選択したい項目を番号で入力
3. 処理完了後にtxtに保存される

## 動作環境
- Python 3.12.10
- 外部ライブラリなし

## ファイル構成
daily_report
├── daily_report_file   #txtファイル収納
└── daily_report.py     #メインプログラム
